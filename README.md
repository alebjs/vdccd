# Venta El Buscón

Sitio web oficial de Venta El Buscón, taberna tradicional en el centro de Madrid.

## Características

- Diseño responsive y moderno
- Información sobre menú y reservas
- Integración con redes sociales
- Optimizado para SEO

## Desarrollo

Este proyecto está construido con:
- React 18
- TypeScript
- Tailwind CSS
- Vite

## Instalación

```bash
npm install
npm run dev
```
